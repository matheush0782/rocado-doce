# 🍰 Roçado Doce

## 📌 Sobre o Projeto
O **Roçado Doce** é uma landing page desenvolvida com o objetivo de digitalizar o atendimento de pequenos negócios locais, especificamente uma produção artesanal de bolo de rolo. A aplicação permite que os clientes visualizem os produtos, escolham quantidades e acompanhem automaticamente o valor total do pedido em tempo real.

---

## 🎯 Problema Resolvido
Muitos pequenos empreendedores recebem pedidos de forma desorganizada através do WhatsApp, o que pode gerar erros, retrabalho e perda de vendas. Este projeto resolve esse problema ao oferecer uma vitrine digital interativa, permitindo que o cliente já informe o que deseja e tenha uma estimativa clara do valor antes de entrar em contato.

---

## 🚀 Funcionalidades
- Exibição de produtos com imagens
- Controle de quantidade com botões (+ e -)
- Limite máximo de 10 unidades por produto
- Cálculo automático do valor total
- Layout responsivo (Mobile First)

---

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
